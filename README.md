# React Foundations

This project is a small Next.js application that accompanies a React course on the Next.js website. It is designed to teach the basics of React that are required to get started with Next.js.

## Course

Visit the official course here:
https://nextjs.org/learn/react-foundations

## What this project covers

- Building UI with Components
- Displaying Data with Props
- Adding Interactivity with State
- Server and Client Components

## Getting started

Install dependencies and start the development server:

```bash
npm install
npm run dev
```

Then open http://localhost:3000 in your browser.
